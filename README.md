# ПВЗ График — Планировщик смен для ПВЗ

## 📱 О приложении
Умный планировщик смен с календарём, расчётом зарплаты, заметками и SVG-аналитикой.

## 📦 Состав проекта
| Файл | Назначение |
|------|-----------|
| `index.html` | Основной файл приложения (1582 строки кода) |
| `manifest.json` | PWA-манифест для установки на телефон |
| `service-worker.js` | Оффлайн-кеширование |
| `privacy.html` | Политика конфиденциальности для RuStore |
| `icons/icon-512.png` | Иконка приложения 512×512 |
| `icons/icon-192.png` | Иконка приложения 192×192 |
| `icons/splash-icon.png` | Фон для splash screen |

## 🚀 Инструкция по конвертации в APK

### Способ 1: PWABuilder (рекомендую)
1. Залей папку `pvz_graphic` на **GitHub Pages** или **Netlify** (бесплатно)
2. Зайди на **pwabuilder.com**
3. Введи URL твоего сайта
4. Нажми **Generate Package → Android**
5. Скачай готовый APK/AAB

### Способ 2: html-to-apk.fast-page.org (проще всего)
1. Упакуй всю папку в ZIP
2. Зайди на сайт **html-to-apk.fast-page.org**
3. Загрузи ZIP
4. Скачай готовый APK

### Способ 3: AppsGeyser
1. Зайди на **appsgeyser.com**
2. Выбери "HTML to APK"
3. Вставь HTML-код
4. Скачай APK

## 🔥 Новые возможности (добавлены)
✅ **PWA-манифест** — телефон предлагает "Добавить на главный экран"
✅ **Service Worker** — работа без интернета
✅ **Splash Screen** — красивая заставка при загрузке
✅ **Офлайн-баннер** — уведомление если нет интернета
✅ **Политика конфиденциальности** — для публикации в RuStore

## 📱 Публикация в RuStore
1. Зарегистрируйся на **console.rustore.ru**
2. Добавь приложение → загрузи APK
3. Заполни описание, загрузи скриншоты
4. Вставь ссылку на политику конфиденциальности
5. Отправь на модерацию
